SeedCup-Competition

requirement:

* python3
* matplotlib
* numpy
* pandas
* pytorch
* sklearn


运行说明：


* 将csv文件"均衡.csv"和data文件"真测试集SVM.data"文件和代码文件放在同一个目录下
* 将module文件夹放在和代码同一个目录下
* 安装python3
* 在命令行中，使用pip install方法安装requirement中的支持包
* 运行HopfieldClassifier.py文件
* 每个训练周期与测试周期返回此次测试的准确率，以及在真测试集下的测试结果
